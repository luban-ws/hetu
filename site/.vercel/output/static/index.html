Git visualization,

Hetu (河图) is a native desktop Git client that renders your branch history as a beautiful, intuitive subway map — with JIRA and CI integration built in.

Download View Source

Everything you need in a Git client

Designed for developers who want clarity and speed — not just another terminal wrapper.

Subway-Map Visualization

Branches and commits rendered as an intuitive metro map. See the entire history at a glance — merges, rebases, and parallel work all clearly visible.

Inline Diff Viewer

Review changes without leaving the app. Toggle between hunk and file views with syntax-highlighted diffs and clear addition/removal markers.

JIRA Integration

Native JIRA panel shows issue details, subtasks, and comments alongside your commits. Link work items directly from your Git workflow.

CI Build Status

AppVeyor build results integrated per-commit. See pass/fail indicators right on the map with on-demand log downloads.

Built with

Tauri 2 Rust Angular git2-rs TypeScript
