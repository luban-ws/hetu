Release Notes 0.4.0

Highlights

  • Fine Tuned Staging And Unstaging - New to explorasa-git is the ability to stage by hunk and lines individually so making partial commit is never easier. File changes view now also supports live update so you can preview changes in realtime.
  • Repo History And Initialization - Managing multiple repository? No problem. explorasa-git now display all previous opened repo conveniently in the file tab so you can switch between repositories easily. You can also initialize a brand new repo just with explorasa-git.
  • JIRA Issue Navigation - Now you can navigate between different JIRA issues with ease. explorasa-git remembers previous JIRA issues you have opened so you don't have to remember the previous story keys.

New

  • Auto Cache Cleanup
  • New JIRA Resolution Selector

Bugfixes

  • #111 File In Changes Sometimes Shows All Line As Created
